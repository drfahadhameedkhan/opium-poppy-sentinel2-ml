# Data Directory

This directory describes how to obtain and organise all datasets required
to reproduce the analysis in:

> **Khan, F.H. (2025).** Geospatial Machine Learning for Opium Poppy Cultivation
> Monitoring in Pakistan and Afghanistan: A Sentinel-2 Multi-Temporal Analysis.
> *Land* (MDPI). ISSN 2073-445X.

---

## Directory Structure

```

```

## Data Sources

### 1. Sentinel-2 Satellite Imagery (Primary Data)

| Item | Detail |
|---|---|
| **Platform** | Sentinel-2A/2B, Multi-Spectral Instrument (MSI) |
| **Product** | Level-2A (Surface Reflectance) |
| **Spatial resolution** | 10 m (VIS/NIR), 20 m (Red Edge/SWIR) |
| **Temporal coverage** | October 2022 – May 2025 (3 seasons) |
| **Scene count** | 4,872 tiles |
| **Source** | Copernicus Open Access Hub / Google Earth Engine |
| **Access** | **FREE** |
| **URL** | https://scihub.copernicus.eu |
| **GEE Collection** | `COPERNICUS/S2_SR_HARMONIZED` |

**→ Acquired automatically by `src/preprocessing/sentinel2_download.py`**

---

### 2. UNODC Annual Opium Survey Data (Validation)

| Item | Detail |
|---|---|
| **Provider** | United Nations Office on Drugs and Crime |
| **Coverage** | Afghanistan (2022, 2023, 2024) |
| **Use** | Ground-truth validation benchmark |
| **Access** | **FREE** |
| **URL** | https://dataunodc.un.org |

Download relevant reports and extract field campaign GPS points.

---

### 3. SRTM Digital Elevation Model (Terrain Features)

| Item | Detail |
|---|---|
| **Product** | SRTM 1 Arc-Second Global (30 m) |
| **GEE Dataset** | `USGS/SRTMGL1_003` |
| **Use** | Elevation, slope, aspect, TWI, TRI derivation |
| **Access** | **FREE** via GEE |

---

### 4. Training Labels (Reference Dataset)

Reference labels compiled from three sources:

1. **UNODC Survey GPS points** — Afghanistan field campaign data (2022–2024)
2. **WorldView-3 expert interpretation** — 0.5 m VHR imagery, stratified subsample
3. **Pakistan ANF verification data** — KPK and Balochistan provincial survey points

Total: **38,417 labelled pixels** across five classes:

| Class | Count | % |
|---|---|---|
| Opium Poppy | 9,842 | 25.6% |
| Wheat | 14,263 | 37.1% |
| Bare/Fallow | 7,816 | 20.3% |
| Other Vegetation | 4,218 | 11.0% |
| Built/Water | 2,278 | 5.9% |

> **Note:** Full reference datasets will be shared upon paper acceptance via
> a Zenodo archive. Contact the author for pre-publication access.

## How to Access Data via Google Earth Engine

```python
import ee
ee.Initialize()

# Sentinel-2 collection
s2 = (ee.ImageCollection("COPERNICUS/S2_SR_HARMONIZED")
      .filterDate("2022-10-01", "2023-05-31")
      .filter(ee.Filter.lt("CLOUDY_PIXEL_PERCENTAGE", 20)))

# SRTM elevation
srtm = ee.Image("USGS/SRTMGL1_003")
slope = ee.Terrain.slope(srtm)
```
